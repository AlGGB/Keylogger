# Keylogger
#Python Keylogger - Research Project

This project consists of a Keylogger developed in Python for educational and research purposes on client-side security (Endpoint Security). The goal is to understand how keylogging attack vectors operate and how antivirus/EDR detect this type of activity.

##  Disclaimer
This script has been created exclusively for educational and ethical audit purposes. Use of this tool against targets without prior written consent is illegal. The author is not responsible for any misuse of this software.

## Characteristics
- Recording keystrokes in real time.
- Local storage in encrypted text file (or optional).
- Library management: `pynput`.

## Requirements
-Python 3.x
- Libraries: `pip install pynput`
